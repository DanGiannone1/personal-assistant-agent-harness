# UI/UX boundary

> **Authority:** Focused current-boundary note.

The supported top-level user surfaces are Engagements, the private Home/Tasks/Calendar/Reminders "My work" pages, Assistant, and Settings. Engagements supports the manual create/open path used in the MVP demo and remains the default landing surface. The Assistant appears as an embedded conversation and a dedicated Assistant route; both operate on the product state through structured outcomes.

The repository does not claim current browser verification, accessibility conformance, or release readiness from source alone. Responsive, keyboard, screen-reader, color-contrast, and browser behavior need focused live testing before they can be asserted. Do not reintroduce global Library/Search, quick-link, or generic workbench claims.
