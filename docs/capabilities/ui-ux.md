# UI/UX boundary

> **Authority:** Focused current-boundary note.

The supported top-level user surfaces are Engagements, Assistant, and Settings. Engagements supports the manual create/open path used in the MVP demo. The Assistant appears as an embedded conversation and a dedicated Assistant route; both operate on the product state through structured outcomes.

The repository does not claim current browser verification, accessibility conformance, or release readiness from source alone. Responsive, keyboard, screen-reader, color-contrast, and browser behavior need focused live testing before they can be asserted. Do not reintroduce personal home, task/calendar/reminder, Library/Search, quick-link, scheduler, or generic workbench claims.
