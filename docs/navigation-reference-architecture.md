# Navigation Design Path Moved

> **Authority:** Compatibility pointer only; superseded 2026-07-14 by issue #15.

The canonical navigation contract is [capabilities/navigation.md](capabilities/navigation.md). It
separates direct host routing from a small typed assistant destination catalog and accepts route
effects only from validated, correlated structured events. Text parsing, marker protocols, and the
former embedding/search picker are not application-control paths.
