## Authority rules

Each rule has one canonical home. A summary, prompt, template, record, or
runtime wrapper may point to that home but must not restate it as a rival rule.
When sources conflict, stop and obtain a decision; do not blend them.

Living guidance has stable names and is updated in place. A dated record captures
evidence, a decision, or a past state and is never treated as current guidance
without re-verification. Archive superseded guidance rather than leaving it
ambiguous. A living document that describes changed behavior is updated in the
same reviewed change.
