# Repository Agent Entrypoint Template

Replace bracketed placeholders when installing this file. This template is not
active until placed in a repository's recognized agent-instruction location.

Before repository work, read `[local documentation index]` and the installed
canonical documents it names: `[Master SDLC]`, `[Testing Charter]` when doing
test work, `[Engineering Operating Standards]`, and `[Agentic Design]` when
changing agents, skills, or handoffs. Read any additional local guidance needed
for the affected architecture, data, security, and delivery surface.

Follow the installed Master SDLC's explicit human-approval gate before
implementation. Keep native runtime loading, tools, permissions, and execution
mechanics in this entrypoint or other native configuration; do not duplicate
shared governance here. If a required source is missing or conflicts, stop and
report it.
